# PayFlow — сопровождение электронных платежей

Минималистичное Flask-приложение для управления электронными платежами.

## Запуск

```bash
cd payment_app
pip install -r requirements.txt
python app.py
```

Открыть: http://127.0.0.1:5000

## Доступы по умолчанию

| Роль  | Логин | Пароль   |
|-------|-------|----------|
| Admin | admin | admin123 |

Новые пользователи регистрируются самостоятельно и получают стартовый баланс 1000 ₽.

## Структура

```
payment_app/
├── app.py               # Flask-приложение
├── requirements.txt
├── payments.db          # SQLite (создаётся автоматически)
└── templates/
    ├── base.html        # Базовый шаблон
    ├── login.html       # Страница входа
    ├── register.html    # Регистрация
    ├── dashboard.html   # Личный кабинет
    ├── send_payment.html # Форма платежа
    └── admin.html       # Панель администратора
```

## Функциональность

- Регистрация / вход по логину и паролю
- Личный кабинет: баланс, история транзакций
- Отправка платежа другому пользователю
- Панель администратора: просмотр всех пользователей и транзакций, пополнение баланса, удаление пользователей
